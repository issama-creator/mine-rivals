# Mine Rivals — план усиления (хит)

## Оценка идеи

Идея сильная: гонка за драгоценности в шахте с вором — сразу понятный конфликт и эмоция («не дай украсть»).  
Вертикальный раннер + соперник лучше, чем просто «лови падающее».

**Уже тянет в сторону хита:**
- ясные роли (ты vs вор)
- смена биомов
- рары vs монеты/бомбы
- обгон

**Ядро, которое нельзя размыть:** вор крадёт только драгоценности. 50 уровней = вариации давления, не смена обоев.

---

## Геймплей / эмоция

1. **Личность вора** — фразы при краже/обгоне, 2–3 анимации («дразнилка», «удрал»). Запоминается сильнее нового коридора.
2. **Риск/награда** — редкий кристалл рядом с динамитом; жадность vs осторожность.
3. **Near-miss juice** — «почти украл / почти поймал» (звук + лёгкий shake). Мобилки любят это.
4. **Boss-уровень каждые 10** — вор агрессивнее / двойной магнит / туннель с ловушками. Даёт «главу», не кашу из 50 одинаковых.

---

## Удержание

1. **3 звезды с разными условиями** — не только «дойди»: `0 бомб`, `лидируй 80%`, `вор < 3 раров`.
2. **Daily** — тот же уровень, другой seed — заход каждый день без 50 новых ассетов.
3. **Сезоны по 10 уровней** — «Акт Лед», награда скина корзины. Легче продавать и обновлять.

---

## Полировка под стор

1. **Трейлер 15–20 сек** — кража рубина + обгон + boom. Иконка: вор тянет кристалл из корзины.
2. **Первый уровень как туториал без текста** — форсированная кража → «ДОГОНИ».
3. **Читаемость спрайтов** — у каждого уровня свой алмаз/бомба/динамит по силуэту, не только цвет.

---

## Монетизация (честная)

1. **Косметика > pay-to-win** — шляпы, корзины, следы пыли.
2. **Continue** за реклу/монеты — один раз на уровне, не бесконечно.
3. **Battle pass** на скины акта, не на силу магнита.

---

## Тех / контент-пайплайн

1. **Пул из модулей** (коридор + таблица спавна + AI вора + win rule) → 50 уровней без 50 уникальных карт.
2. **Реплей последних 5 сек** при проигрыше — «вот где украли». Бесит → мотивирует рестарт.

**Самый жирный ROI:** характер вора + 3-звездные цели + boss каждые 10 + туториал-кража на 1 уровне.

---

## Что добить, чтобы чувствовался хит

1. **Один ясный хук за 3 секунды** — «обогони вора и не дай украсть кристаллы». Меню/первый уровень должны кричать это без текста.
2. **Короткий раунд + «ещё раз»** — 50 уровней ок, но каждый 45–90 сек (не длинный марафон).
3. **Прогрессия наград** — звёзды / сундук / скин корзины после уровня. Без мета люди отвалятся к 10-му.
4. **Читаемость** — свой набор спрайтов на уровень (алмаз ≠ монета ≠ динамит).
5. **Вор как характер** — насмешки, уникальные анимации кражи, иногда «подстава».
6. **Звук/haptic** — сейчас почти пусто; для хита это ~30% ощущения.
7. **ASO** — название, иконка «вор тянет кристалл», 30-сек трейлер с кражей.

## 50 уровней — как сделать правильно

Не 50 копий с другой картинкой. Лучше **10 биомов × 5 миссий**:

| Слой | Пример |
|------|--------|
| Коридор | лёд / лес / лава / грибы… |
| Цель | набери 8 раров / не дай вору 5 / добеги с лидом |
| Угроза | бомба / динамит (радиус) / скользкий пол |
| Элементы уровня | только «свои» камни и ловушки |

**Правила на уровень:**
- 1 новый элемент или механика (не 5 сразу)
- свой пул лута (этот шахтный изумруд, этот динамит)
- коридор из 2–3 тайлов на биом, не 50 уникальных гигантских PNG
- длина ~800–1200 м (или короче для детей), скорость растёт к концу акта

**Структура контента:**
```
levels/01_crystal.json   → corridor, spawnTable, winRule, thiefAggression
assets/corridors/ice_a.png, ice_b.png
assets/items/level_01/...
```

## Приоритет на ближайшие шаги

1. Укоротить забег + экран «уровень N / 50» *(частично сделано: короткий забег + простой UI)*
2. JSON-уровни + свой spawn table
3. 5 биомов × 3 уровня = прототип «сезона»
4. Динамит как отдельная угроза (не клон бомбы)
5. Мета: 3 звезды, скин, daily challenge
6. Разнообразие коридоров: **свои элементы на каждый коридор**
