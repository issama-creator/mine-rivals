# Dino Rivals — финальный геймплей + V1 scope

Документ-замок: **так играется игра.**  
Не спорить каждую неделю. Новые идеи — в бэклог, не в core.

---

## Финальный геймплей (одной картиной)

Ты **украл яйцо динозавра**. Сзади бегут **два динозавра**.  
Ты **автоматически бежишь** вперёд по **3 дорожкам**, рулишь только **влево/вправо**.

**Цель рана:** донести яйцо до **базы (финиш)**.

**Проигрыш:** дино **догоняют** (дистанция преследования = 0).

**Как не дать себя поймать:** **меньше ошибайся.**  
Любая ошибка (столкновение, ловушка, сильное замедление) → дино **сокращают дистанцию**.  
Чем ближе они — тем **громче шаги, рев, сильнее тряска**. Опасность **слышно**, даже без HUD.

**По пути** подбираешь **кристальные яйца** — это **валюта** магазина (скины, бусты).

**Добежал до финиша:**
1. **Egg Rescued!**
2. Открывается **сундук** → выпадает **рандомное коллекционное яйцо** (редкость видна).
3. Кристальные яйца с рана **капают на счёт**.

**Поймали по дороге:**
- Сундука коллекции **нет**.
- Кристальные яйца с рана **всё равно частично/полностью начисляются** (ран не впустую).

**Между ранами:** тратишь кристальные яйца на скины и 1–2 буста перед стартом (щит / адреналин).

**Эмоция:** не «собери больше, чем вор», а **«слышу их за спиной — успею ли до базы»**.

---

## Design locks (не ломать)

1. Core = ошибки → gap ↓ → поймали / финиш.  
2. USP = **звук + тряска** от близости дино + яйцо в руках.  
3. Валюта с пути ≠ коллекция. Коллекция **только из сундука на финише**.  
4. Смерть **не обнуляет** весь фарм валюты.  
5. Не тащить gambling-checkpoint из Mine Rivals в Dino.  
6. Бустов в V1 максимум **два**.  
7. Один «давление» gap, даже если дино визуально двое.

## Режимы

| Режим | Что это | Ощущение |
|-------|---------|----------|
| **Adventure** | **~5 км**, 2 дино | основной хит |
| **Hardcore** | те же правила, **злее** (быстрее темп и/или сильнее штраф за ошибку). Длина: **~5–8 км** в V1; **10 км** — после того как 5 км уже кайф |

> Hardcore = сложнее, не обязательно «в 2 раза длиннее ради длины».

---

## Две награды (чётко)

| | Кристальные яйца | Коллекционное яйцо |
|--|------------------|---------------------|
| Где | пикапы **по пути** | **сундук только на финише** |
| Зачем | магазин | коллекция / «ещё один ран за редкость» |
| При смерти | **да, на счёт** (V1: 100% или 70%) | нет |

На финише сундук = **один джекпот**: коллекция (+ редкость) + банк кристальных с рана. Без лишнего лута.

## V1 IN (2–3 недели)

### В ране
- 3 lane, автобег, рост скорости
- Сюжетное яйцо на герое
- 2 дино сзади + gap от ошибок
- 4–5 препятствий + пикап кристальных яиц
- Звук/shake от близости
- Финиш базы → Egg Rescued → сундук

### Мета
- Кошелёк кристальных яиц
- Магазин: 2–3 скина, бусты Shield + Adrenaline
- Коллекция: 3–5 яиц с редкостью (common → rare → legendary)
- 1 локация (вторая — опционально)

### UI
- Меню: Adventure / Hardcore, магазин, коллекция
- HUD минимальный (счётчик кристальных в ране)
- Results: поймали vs финиш+сундук

## Одной фразой для команды и стора

**Steal the egg. Don't get caught.**  
Донеси до базы, копи кристальные яйца по пути, на финише открой сундук коллекции — и всё время слышь дино за спиной.



Звуки дино обычно берут так:

Бесплатно (для старта)

Freesound.org — рев, топот, дыхание (смотри лицензию CC0 / CC-BY)
Mixkit
OpenGameArt
Платно (чище для стора)

Epidemic Sound / Artlist — подписка, удобно для релиза
Soundsnap
Pond5 / AudioJungle — разовые SFX «dinosaur roar», «heavy footsteps»
Собрать самому (часто лучший «dino»)

Топот: низкий punch + low rumble + лёгкий dirt
Рев: животное (тигр/слон/крокодил) вниз по тону + reverb
В Audacity / Reaper — 2–3 слоя = «свой» дино, меньше похоже на сток
ИИ (осторожно)

ElevenLabs / другие SFX-генераторы — ок для прототипа; для стора проверь права тарифа
Для V1 хватит:
1 топот (loop по громкости от gap), 2–3 рева, 1 near-miss snarl. Не библиотека на 100 файлов.

Итого: прототип — Freesound/Mixkit; под релиз — подписка SFX или свои слои из стока.
